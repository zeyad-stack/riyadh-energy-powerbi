# ⚡ Riyadh Energy Consumption Dashboard (2005–2040)

This Power BI project analyzes electricity consumption trends in Riyadh using actual and forecast data.

## 📊 Key Insights
- Total consumption expected to exceed **300 million MWh by 2040**.
- Compound annual growth rate (CAGR): **~3.6%**
- Temporary dip between **2018–2020**, followed by recovery.

## 🧩 Recommendations
- 🌞 Invest in renewable energy.
- 🏙️ Improve building energy efficiency.
- ⚙️ Develop smart grids to optimize distribution.
- 🔋 Support energy storage and demand management programs.

## 🛠️ Tools Used
Power BI · DAX · Excel · Data Visualization
